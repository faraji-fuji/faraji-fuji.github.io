Heal Flex

Heal Flex is a web application that recommends users to appropriate therapists. The application compares the user’s preferences to available therapists and recommends the best match. The user may then proceed to book a session and pay relevant fees. For example, a user with issues a, b and c may want a therapist with attributes e, f and g. The applications on the backend checks to see which of the therapists can deal with issues a, b and c and has attributes e, f and g. The application then recommends the top 3 therapists that match.

Insta-clone

This is a simplified replica of Instagram where users can create posts. All posts have an image attached to it (PNG or JPG only) and some text associated with it. There is a system here where users can follow each other to see their posts. Hence a timeline where the posts of a user and the posts of users they follow are merged together. Users are permitted to comment on other posts. The main task in this project is to deal with and manage multiple data storage mechanisms. The Google Cloud Storage Bucket is solely used for managing images whereas the Google Datastore is used to manage all other information.
